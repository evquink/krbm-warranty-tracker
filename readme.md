# KRBM Warranty Tracker

Custom HTML form for submitting warranty and guarantee records to Supabase database.

## Features
- Searchable property dropdown
- Client-side validation
- Email domain restriction (@keyrenterbuxmont.com)
- Direct Supabase integration via REST API
- Responsive design (mobile/tablet/desktop)

## Tech Stack
- Frontend: Vanilla HTML/CSS/JavaScript
- Backend: Supabase (PostgreSQL + REST API)
- Hosting: Railway.com (Express.js static server)

## Deployment
This app is deployed on Railway.com and serves a single HTML file with embedded JavaScript.
